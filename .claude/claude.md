## 标准入库流程 (7 步)

### Step 1: 读取论文
- Word 文档 -> Read 工具直接读取
- PDF 文件 -> Read 工具读取
- DOI/URL -> WebFetch 抓取摘要

### Step 2: 按维度提取信息
严格按照  树状图的 8 大分支逐项提取：
1. 论文元信息 2. 原料体系 3. 工艺技术 4. 产物体系
5. 工程放大 6. 技术经济 7. 商业化 8. 论文间关联

### Step 3: 生成结构化 JSON
按  输入格式组装 JSON。

### Step 4: 生成 Markdown 分析报告
按  模板生成报告。

### Step 5: 自动比对已有论文库
按 7 条关联检测规则匹配已有论文。

### Step 6: 输出关联报告

### Step 7: 交付物
1. Markdown 分析报告 2. JSON 文件 3. 关联检测结果
4. 入库命令: [ERROR] 文件不存在: paper-data-XXX.json
