# 滑石粉/硫酸镁 论文商业化分析体系 · 基础架构

> 本体系用于系统性分析滑石粉（硅酸镁）相关论文，评估其商业化潜力，并建立论文间关联网络。
> 数据库采用**混合模式**：核心高频字段用宽表，低频/小众字段用 EAV 扩展表。

---

## 分析维度树状图

```
📁 论文分析体系根节点
│
├── 📁 1. 论文元信息 (Metadata)
│   ├── 1.1 基础标识
│   │   ├── 论文标题 (title)
│   │   ├── 作者 / 机构 / 国家
│   │   ├── 发表年份 / 期刊 / DOI
│   │   └── 研究类型 (综述 / 实验 / 模拟 / 中试 / 工业化)
│   ├── 1.2 研究背景
│   │   ├── 解决的问题陈述
│   │   ├── 原料来源 (滑石矿产地、品位、伴生矿)
│   │   └── 对标技术路线 (该论文试图替代/改进的现有工艺)
│   └── 1.3 可信度标记
│       ├── 是否有实验数据支撑
│       ├── 是否有经济性数据
│       ├── 实验规模 (实验室 / 小试 / 中试 / 工业化)
│       └── 引用质量与被引次数
│
├── 📁 2. 原料体系 (Raw Material System)
│   ├── 2.1 原料规格
│   │   ├── 滑石粉纯度 / 品位 (MgO含量、SiO₂含量)
│   │   ├── 杂质种类与含量 (Fe₂O₃, Al₂O₃, CaO, 烧失量)
│   │   ├── 粒度要求 (目数 / D50 / 比表面积)
│   │   └── 白度 / 其他物理指标
│   ├── 2.2 原料预处理
│   │   ├── 是否需要预处理 (是 / 否)
│   │   ├── 预处理方式 (粉碎/分级/酸洗/浮选/磁选/煅烧预处理)
│   │   ├── 预处理成本估算
│   │   └── 预处理产率损失
│   ├── 2.3 辅助原料
│   │   ├── 所需试剂清单 (酸/碱/盐/有机溶剂/催化剂)
│   │   ├── 各试剂用量 (吨试剂 / 吨产品)
│   │   ├── 试剂价格与来源可得性
│   │   └── 水/电/气消耗 (吨或kWh / 吨产品)
│   └── 2.4 原料约束评估
│       ├── 是否有地域限制 (如某地矿石才适用)
│       ├── 供应链稳定性
│       └── 原料替代可能性
│
├── 📁 3. 工艺技术体系 (Process Technology)
│   ├── 3.1 技术路线分类 (大类)
│   │   ├── A. 物理法 (粉碎/分级/表面改性/浮选提纯)
│   │   ├── B. 热化学法 (煅烧/热解/焙烧 → MgO + SiO₂)
│   │   ├── C. 湿法冶金—酸法 (HCl/H₂SO₄/HNO₃浸出 → 镁盐)
│   │   ├── D. 湿法冶金—碱法 (NaOH/Na₂CO₃沉淀)
│   │   ├── E. 碳化法 (CO₂加压碳化 → Mg(HCO₃)₂ → MgCO₃)
│   │   ├── F. 电化学法 (电解/电渗析)
│   │   ├── G. 生物法 (微生物浸出)
│   │   └── H. 组合/耦合工艺 (如 酸浸+碳化、煅烧+酸浸)
│   ├── 3.2 核心工艺参数
│   │   ├── 反应温度范围 & 最优温度
│   │   ├── 反应压力 (常压/加压，具体数值)
│   │   ├── 反应时间 / 停留时间
│   │   ├── 酸浓度/碱浓度/液固比
│   │   ├── 搅拌速率
│   │   ├── pH 控制范围
│   │   └── 气氛要求 (空气/N₂/CO₂/真空)
│   ├── 3.3 产率链 (从原料到产品的每一步)
│   │   ├── 步骤1: [名称] → 转化率 ___%
│   │   ├── 步骤2: [名称] → 转化率 ___%
│   │   ├── ...
│   │   ├── 总产率 (从滑石粉到最终产品)
│   │   └── 产率瓶颈步骤识别
│   ├── 3.4 设备体系
│   │   ├── 核心反应器类型 (搅拌釜/回转窑/流化床/高压釜/电解槽)
│   │   ├── 关键设备材质要求 (耐酸/耐碱/耐高温)
│   │   ├── 分离设备 (过滤/离心/沉降/膜分离)
│   │   ├── 干燥设备 (喷雾干燥/回转干燥/闪蒸干燥)
│   │   ├── 设备国产化率 / 进口依赖度
│   │   └── 设备投资估算 (万元 / 万吨产能)
│   ├── 3.5 能耗体系
│   │   ├── 热能消耗 (蒸汽 / 燃煤 / 天然气，折标煤吨/吨产品)
│   │   ├── 电能消耗 (kWh / 吨产品)
│   │   ├── 各步骤能耗占比
│   │   └── 余热回收潜力
│   └── 3.6 环保与副产品
│       ├── 废气 (种类/排放量/处理方式/处理成本)
│       ├── 废水 (种类/排放量/处理方式/处理成本)
│       ├── 废渣 (种类/排放量/是否可资源化利用)
│       ├── 副产品 (SiO₂残渣/铁红/CaSO₄等，是否有销售价值)
│       └── 是否满足当前环保政策 (固废法/碳达峰等)
│
├── 📁 4. 产物体系 (Product System)
│   ├── 4.1 主产品定义
│   │   ├── 产品化学名称与分子式
│   │   ├── 产品类别 (氧化镁/氢氧化镁/碳酸镁/硫酸镁/氯化镁/
│   │   │              金属镁/白炭黑/硅酸钠/硅胶/水滑石类/...)
│   │   └── 产品形态 (粉末/晶体/溶液/浆料)
│   ├── 4.2 产品质量指标
│   │   ├── 主含量 / 纯度 (%)
│   │   ├── 杂质含量上限 (各关键杂质)
│   │   ├── 粒度 / 比表面积 / 堆积密度
│   │   ├── 白度 / 色度
│   │   ├── 是否符合国标/行标/国际标准
│   │   └── 与市售同类产品的质量对比
│   ├── 4.3 产品市场定位
│   │   ├── 该纯度的市场应用场景 (如 ≥99% MgO用于...)
│   │   ├── 该纯度对应的市场价格 (元/吨)
│   │   ├── 高纯品溢价空间 (纯度每提高1%价格提升多少)
│   │   └── 市场规模与需求增长率
│   ├── 4.4 副产品价值
│   │   ├── 副产品可否销售
│   │   ├── 副产品市场单价
│   │   └── 副产品收益对整体经济性的贡献率
│   └── 4.5 产品组合策略
│       ├── 是否可以灵活切换产品 (同一产线产多种镁化合物)
│       └── 多产品联产的经济优势
│
├── 📁 5. 工程放大评估 (Scale-up Assessment)
│   ├── 5.1 当前成熟度
│   │   ├── TRL 等级 (1-9)
│   │   ├── 当前规模 (g级 / kg级 / 吨级 / 万吨级)
│   │   └── 放大到工业级的跃迁倍数
│   ├── 5.2 放大风险
│   │   ├── 传热问题 (放大后散热/加热是否可行)
│   │   ├── 传质问题 (混合/接触效率)
│   │   ├── 腐蚀问题 (设备材质是否满足工业生产寿命)
│   │   ├── 堵塞/结垢风险
│   │   └── 安全风险 (高温高压/有毒试剂/粉尘爆炸)
│   ├── 5.3 放大路径
│   │   ├── 建议放大步骤 (小试→中试→示范→工业化)
│   │   └── 每步所需时间与投资
│   └── 5.4 同类技术工业化先例
│       ├── 是否有类似工艺已工业化
│       └── 工业化案例的规模与运营情况
│
├── 📁 6. 技术经济分析 (Techno-economic Analysis)
│   ├── 6.1 投资估算 (CAPEX)
│   │   ├── 论文是否提供投资数据 (有/无/可推算)
│   │   ├── 设备投资 (万元)
│   │   ├── 土建投资 (万元)
│   │   ├── 安装工程费 (万元)
│   │   └── 单位产能投资 (万元 / 万吨年产能)
│   ├── 6.2 运营成本 (OPEX)
│   │   ├── 论文是否提供成本数据 (有/无/可推算)
│   │   ├── 原料成本 (元 / 吨产品，占比___%)
│   │   ├── 能耗成本 (元 / 吨产品，占比___%)
│   │   ├── 人工成本
│   │   ├── 折旧
│   │   └── 环保处理成本
│   ├── 6.3 盈利分析
│   │   ├── 吨产品总成本 (元/吨)
│   │   ├── 吨产品售价 (元/吨)
│   │   ├── 吨产品毛利 (元/吨)
│   │   ├── 毛利率
│   │   ├── 投资回收期 (年)
│   │   └── IRR / NPV (若论文提供)
│   ├── 6.4 盈亏平衡与敏感性
│   │   ├── 盈亏平衡产能利用率
│   │   ├── 对原料价格的敏感性 (原料涨10%→利润降__%)
│   │   ├── 对产品价格的敏感性
│   │   └── 对能耗价格的敏感性
│   └── 6.5 与现有工艺的经济性对比
│       ├── 对比对象 (如 海水卤水法/菱镁矿法/白云石法)
│       ├── 成本优势/劣势
│       └── 综合竞争力判断
│
├── 📁 7. 商业化综合评估 (Commercialization)
│   ├── 7.1 价值主张
│   │   ├── 该技术解决的核心商业痛点
│   │   ├── 独特卖点 (成本更低/纯度更高/环保/原料利用)
│   │   └── 目标客户画像
│   ├── 7.2 商业化障碍
│   │   ├── 技术障碍 (哪些参数尚未验证)
│   │   ├── 市场障碍 (市场准入/客户认证周期/竞争格局)
│   │   ├── 资金障碍 (首套装置投资门槛)
│   │   ├── 政策障碍 (环评/安评/能评难度)
│   │   └── 原料障碍 (滑石粉供应稳定性与价格)
│   ├── 7.3 商业化路径建议
│   │   ├── 建议的产业化路径 (自建产线/技术授权/合资/...)
│   │   ├── 建议的首选产品与规模
│   │   ├── 建议的选址考量
│   │   └── 下一步需验证的关键假设
│   ├── 7.4 竞争格局
│   │   ├── 现有主流工艺路线与主要生产企业
│   │   ├── 替代技术威胁
│   │   └── 本技术的竞争位势 (成本领先/差异化/利基市场)
│   └── 7.5 综合商业评分
│       ├── 技术可行性 (0-10)
│       ├── 经济可行性 (0-10)
│       ├── 市场吸引力 (0-10)
│       ├── 放大可行性 (0-10)
│       └── 综合推荐指数 (加权总分)
│
└── 📁 8. 论文间关联网络 (Cross-paper Synergy Network)
    ├── 8.1 关联类型定义
    │   ├── 上下游衔接 (A论文生产中间体 → B论文将其加工为最终产品)
    │   ├── 工艺互补 (A论文的预处理 + B论文的反应路线)
    │   ├── 副产品利用 (A论文的废渣 → B论文的原料)
    │   ├── 条件优化借鉴 (B论文的参数优化思路可用于A论文)
    │   ├── 对标竞争 (A和B生产同一产品，不同路线，可对比)
    │   ├── 设备共用 (A和B可使用同一核心设备)
    │   └── 联合生产 (A+B → 联产多种产品，共享公用工程)
    ├── 8.2 关联记录
    │   ├── paper_id_a ↔ paper_id_b
    │   ├── 关联类型
    │   ├── 关联强度 (强/中/弱)
    │   ├── 关联描述
    │   └── 组合后的商业潜力评分
    └── 8.3 新论文入库自动比对
        ├── 提取关键词自动匹配已有论文
        ├── 标记潜在关联
        └── 生成关联报告
```

### EAV 扩展表部分（灵活字段）

#### 表: `paper_attributes` — 任意扩展属性

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| paper_id | INTEGER FK | 关联 papers.id |
| category | TEXT | 属性大类 (见下方) |
| attr_key | TEXT | 属性名 |
| attr_value | TEXT | 属性值 |
| attr_unit | TEXT | 单位 (可选) |
| source_context | TEXT | 数据来源（论文中的位置/原文引用） |

**category 取值（对应树状图分支）：**
- `raw_material_spec` — 原料规格（2.1）
- `raw_material_pretreatment` — 原料预处理（2.2）

## 使用流程

### 单篇论文入库流程

```
1. 用户提供论文 (PDF / 标题 / DOI)
2. 阅读并提取信息 → 按本树状图逐项填写
3. 生成结构化数据 → 写入 papers + 子表 + EAV
4. 生成 Markdown 可读分析报告 → thesis-analysis/[论文简称].md
5. 自动比对已有论文 → 检测潜在关联
6. 如有关联 → 生成关联报告 + 写入 synergies 表
```

### 多论文查询示例

```sql
-- 查询所有"酸法+硫酸镁"路线且毛利率>30%的论文
SELECT title, tech_route_detail, gross_margin, composite_score
FROM papers
WHERE tech_route_category = 'C'
  AND target_product_name LIKE '%硫酸镁%'
  AND gross_margin > 30
ORDER BY composite_score DESC;

-- 查找可能与某新论文关联的已有论文
-- (基于产品类别 + 技术路线的匹配规则)
SELECT p2.title, p2.target_product_name, p2.tech_route_detail
FROM papers p1, papers p2
WHERE p1.id = [新论文ID]
  AND p1.id != p2.id
  AND (
    p1.target_product_category = p2.target_product_category  -- 同类产品 → 对标竞争
    OR (p1.tech_route_category = 'H' AND p2.tech_route_category IN ('B','C'))  -- 耦合工艺 → 上下游
  );
