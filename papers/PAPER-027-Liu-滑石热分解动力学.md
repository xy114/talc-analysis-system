## 核心贡献：滑石热分解的完整动力学参数 + 物相演化定量数据

以广西龙广高纯滑石（>98 wt%）为模型体系，在 200–1300°C 全温度范围内进行了 XRD + FTIR + TG-DSC 系统表征，并首次以 **Coats-Redfern 积分法** 建立了滑石脱羟基的动力学模型。

### 关键定量数据

| 参数 | 数值 | 方法 |
|------|------|:--:|
| 脱羟基活化能 Ea | **69 kcal/mol (289 kJ/mol)** | Coats-Redfern 积分法 |
| 反应级数 n | **1（一级反应）** | 最佳拟合 (R²→max) |
| 脱羟基起始温度 | **~800°C** | XRD: 顽火辉石首次出现 |
| 脱羟基峰值温度 | **~895°C** | DSC 吸热峰 |
| 总失重 | **~4.3 wt%** | TG |
| 完全脱羟基 | **1000°C** | Mg-OH IR 带消失 |
| 结晶度 (25°C→700°C) | **100% → 32.2%** | (001)面衍射强度指数法 |
| 顽火辉石晶粒 (800→1200°C) | **15.0 → 43.2 nm** | Scherrer 公式 |
| d₀₀₁ (室温→700°C) | **0.94 → 0.92 nm** | 层间距收缩 |

### 两步分解反应

```
Step 1 (~800°C): 2Mg₃Si₄O₁₀(OH)₂ → 3Mg₂Si₂O₆(顽火辉石) + 2SiO₂(无定形) + 2H₂O(g)
Step 2 (~1200°C): Mg₂Si₂O₆(顽火辉石) + SiO₂(无定形) → Mg₂Si₂O₆(斜顽火辉石) + SiO₂·nH₂O(opal-C)
```

## 第一性原理分析

### 活化能 289 kJ/mol 的物理意义

滑石脱羟基活化能 289 kJ/mol 与以下键能对比：

| 化学键 | 键能 (kJ/mol) | 
|------|:---:|
| Mg-O (滑石八面体层) | ~155 |
| Si-O (四面体层) | ~466 |
| O-H (羟基) | ~467 |
| **脱羟基 Ea** | **289** |

Ea=289 kJ/mol 介于 Mg-O 键能(155) 和 O-H 键能(467) 之间，且接近两者的平均值——说明脱羟基过程的速控步不是单纯的 O-H 断裂（需 467 kJ/mol），而是 Mg-OH 配位环境的协同重组（Mg-O 弱化 + O-H 断裂 + H₂O 扩散的耦合过程）。这直接支持了 PAPER-023 的核心结论：脱羟基驱动力不是 O-H 键的热断裂，而是整体结构失稳。

### 一级反应动力学的物理含义

n=1（一级反应）意味着分解速率仅取决于未反应滑石的比例——**每个滑石层的脱羟基是独立事件，不受邻近层脱羟基状态影响**。这与滑石的 2:1 层状结构一致：每个 TOT 层的 OH 位于八面体层内部，层间为范德华力，脱羟基时 H₂O 分子沿层间通道逸出，各层独立反应。
