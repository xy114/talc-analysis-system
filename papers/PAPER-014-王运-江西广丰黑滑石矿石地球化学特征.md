## 核心贡献：矿石类型分类 + 有机碳含量差异 + 资源量确认

通过野外调查 + 岩相学 + 地球化学分析 + 碳含量分类，对广丰黑滑石矿石进行系统分类，为广丰百亿元产业基地提供数据支撑。

### 向 `raw_black_talc` 追加性质数据 (4 条)

| 性质 | 内容 |
|------|------|
| 矿石类型 | 假鲕状滑石岩 / 角砾状滑石岩 / 片理化滑石岩 |
| 有机物含量差异 | 片理化型有机碳含量明显高于假鲕状和角砾状 |
| 广丰资源量 | 累计探获超 **2 亿吨**，超大型矿床 |
| 应用等级 | 初步达到塑料/陶瓷/橡胶/涂料用滑石一级标准 |

### 交叉验证

- 储量数据（2 亿吨）与 PAPER-012（1.05 亿吨广丰 + 2.8 亿吨全国）、PAPER-016（6-8 亿吨全国）交叉参照——数值差异来自统计口径和年份
- 矿石类型分类与 PAPER-013 李成祥 2016（鳞片状/鲕状/变形鲕状）互补
