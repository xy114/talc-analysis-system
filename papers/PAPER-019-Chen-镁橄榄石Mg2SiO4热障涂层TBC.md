## 核心贡献：首次系统验证 Mg₂SiO₄ 作为热障涂层材料

**本论文首次对 Mg₂SiO₄（镁橄榄石）作为燃机热障涂层进行了全系统验证**——从粉体合成到 APS 喷涂到 830+ 次热循环测试到失效机理分析。核心结论：Mg₂SiO₄ 在多项关键指标上**优于工业标准 8YSZ**。

### 与 8YSZ 全面对比

| 性能指标 | **Mg₂SiO₄** | 8YSZ (工业标准) | 优劣 |
|------|:--:|:--:|:--:|
| 相稳定性 | **1573 K 100h 无相变** | 1573 K 出现 c 相 | ✓ 优于 |
| 热导率 (1273 K) | **比 8YSZ 低 ~20%** | 2.1-2.2 W/m·K | ✓ 优于 |
| 热膨胀系数 (473→1623 K) | 8.6→11.3×10⁻⁶ K⁻¹ | ~11×10⁻⁶ K⁻¹ | ≈ 相当 |
| 断裂韧性 | 2.8 MPa·m¹/² | ~3 MPa·m¹/² | ≈ 相当 |
| 硬度 | ~10 GPa | ~12 GPa | ≈ 略低 |
| 抗烧结 (1773 K/6h) | **收缩 13%** | 收缩 18% | ✓ 优于 |
| 热震寿命 (1573 K 水淬) | **~16 次** | ~7 次 | **2×** |
| 密度 | **3.21 g/cm³** | 6.0 g/cm³ | ✓ 轻 45% |
| 涂层热循环寿命 (1273 K) | **>830 次** | — | 可商业化 |
| 原料成本 | 低廉（无稀土） | 含 Y₂O₃（稀土） | ✓ 优于 |
