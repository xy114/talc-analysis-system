## 核心贡献：多产地成分对比 + 滑石颜色分类体系

2014 年产业现状综述，提供国内主要产地滑石化学成分对比数据，建立粉红/白/绿/灰/黑五种颜色分类体系。

### 向 `raw_black_talc` 追加性质数据 (3 条)

| 性质 | 内容 |
|------|------|
| 各地成分对比 | 辽宁海城: SiO₂ 57% MgO 27% CaO 3% LOI 8%；**广西上林: SiO₂ 62% MgO 30% CaO 0.3% LOI 6%（国内最高品位）** |
| 国内滑石颜色分类 | 粉红 / 白 / 绿 / 灰 / 黑滑石五种，黑色因含有机质 |
| 产地列表 | 江西广丰 / 重庆南桐 / 湖南保靖 / 甘肃金塔 / 江西九江 / 宜春 / 萍乡 / 广东平远 / 四川丰都 / 广西上林 / 湖北钟祥 |

### 广西上林高品位滑石

PAPER-017 首次指出广西上林滑石为国内最高品位（MgO 30%, CaO 仅 0.3%）——这对于高纯 MgO 提取路线（如 PAPER-007 的 HF+HCl 酸浸）的原料选择有直接指导意义。
