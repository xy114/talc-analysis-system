## 核心贡献：桃源矿床矿物定量 + 致黑机理确认

光学显微镜 + XRD + EPMA + XRF 多技术联合表征，对广丰桃源矿床进行系统矿物学定量分析。**确认致黑机理为 T-O-T 层间类石墨烯碳层插层**，而非有机质独立颗粒。

### 向 `raw_black_talc` 追加性质数据 (5 条)

| 性质 | 内容 |
|------|------|
| 矿物组成（定量） | 滑石 54.3-71.7% + 白云石 17.7-31.7% + 石英 8.7-14.0% |
| 致黑机理 | T-O-T 层间类石墨烯碳层插层，非有机质独立颗粒 |
| 矿石工业类型 | 沉积改造型，鳞片状滑石岩为主，少量变形假鲕状 |
| 晶体纯度 | 杂质元素较少，接近理想滑石晶体化学式 |
| 煅烧产品等级 | 满足防水材料用滑石粉二级品 / 通用滑石粉三级品 |

### 交叉验证

- 致黑机理与 PAPER-013 李成祥 2016（类石墨烯碳层特征）独立确认一致
- 矿物组成补充了 PAPER-002 的化学成分数据（PAPER-002 给元素含量，本论文给矿物相含量）
- 产品等级数据为工艺设计提供质量标准参考
