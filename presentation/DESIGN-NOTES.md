## 二、学术 PPT 十大设计规则

### 规则 1: 字体层次 — 标题与正文的尺寸比例

| 元素 | 推荐尺寸 | 说明 |
|---|---|---|
| 幻灯片标题 | **32–40 pt** (最小 28 pt) | 会议室应能从屏幕高度 6 倍距离阅读 |
| 正文/要点 | **24–28 pt** (最小 18 pt) | 18 pt 以下投影后不可读 |
| 图注/引用 | **14–16 pt** | 仅用于次要信息 |
| 表格内文字 | **18–24 pt** | 须保证最远观众能看清 |

**核心比例**: 标题与正文的比值建议在 **1.3:1 至 1.5:1** 之间。过大差距会破坏视觉连续性。

### 规则 2: 幻灯片密度 — "一页一观点" + 6×6 规则

- **每页不超过 6 个要点，每个要点不超过 6 个词**（6×6 规则）
- **一页只讲一个核心观点** — 若需要多个观点，拆分为多页
- **7 秒测试**: 观众应在幻灯片出现后 7 秒内抓住核心信息
- **节奏**: 每页约 45–60 秒讲解时间

### 规则 3: 色彩克制 — 科学演示的色彩原则

- **每套 PPT 控制在 2–4 种颜色**（含背景）
- **策略**: 1 种主色 + 1 种强调色 + 中性色（灰/白/黑）
- **对比度**: 文本与背景对比度须 ≥ **4.5:1**（WCAG AA），推荐 ≥ **7:1**（WCAG AAA）
- **禁止只靠颜色区分信息**: ~8% 的男性观众有红绿色盲。须同时使用线型、标记形状、标签
- **灰度测试**: 幻灯片转为黑白后应仍可区分所有内容
- **Okabe-Ito 色盲友好色板**: #0173B2 (蓝), #DE8F05 (橙), #029E73 (绿), #D55E00 (朱红)

### 规则 4: 数据图表注释规范

- **坐标轴**: 必须标注 "变量名 (单位)"，不可遗漏单位
- **面板标签**: 大写字母 (A, B, C...)，粗体，9–10 pt，置于左上角
- **图例**: 优先直接在线旁标注曲线，其次才是独立图例
- **误差棒**: 有数据点时必须显示变异性（SD/SEM/CI）
- **标注**: 用箭头和文本框引导观众至关键特征
- **禁止**: 3D 图表、饼图、回归方程显示在图上、网格线（除非必要）
- **自解释**: 图表应不读正文也能理解

### 规则 5: 表格样式 — "三线表"标准

| 要素 | 推荐 | 说明 |
|---|---|---|
| 顶线 | **1.5 pt** (最粗) | 表格最上端 |
| 表头下线 | **0.75 pt** (中等) | 列标题下方 |
| 底线 | **1.5 pt** (最粗) | 表格最底部 |
| 竖线 | **无** | 学术表格不画竖线 |
| 表体底色 | **无填充** | 避免色块（除非有科学含义） |
| 表标题 | **位于表格上方** | 与图标题（下方）区分 |

### 规则 6: 图标题格式

- **位置**: 图标题在**图下方**；表标题在**表上方**
- **编号**: "Figure 1", "Figure 2"...（不要用 "Graph 1"）
- **字号**: 比正文小 1 pt
- **自包含**: 标题需完整描述"什么变量、什么条件、样本量 n=X、统计信息"
- **不要写**: "Graph of..."、"See text for explanation"、"as shown below"

### 规则 7: 字体选择 — 投影专用

- **正文必须使用无衬线字体（sans-serif）**: Calibri, Arial, Helvetica, Open Sans, Roboto
- **标题可用衬线字体做变化**，但正文必须 sans-serif
- **最多使用 1–2 种字体家族**
- **避免**: 斜体（投影难读）、下划线（像超链接）、全大写正文、居中对齐正文
- **行距**: ≥ **1.2 倍**，推荐 1.5 倍

### 规则 8: 背景与对比——亮底 vs 暗底

| 场景 | 推荐 | 理由 |
|---|---|---|
| 小教室 (<50人) | **亮底深字** | 印刷友好，阅读舒适 |
| 大会议室 (>50人) | **暗底亮字** | 发光效果，后排清晰 |
| 材料科学数据图 | **白底** | 与期刊风格一致，便于引用 |

- **避免渐变背景**（除非有功能目的）
- **避免模板中无意义的装饰图形**

### 规则 9: 图片与图表分辨率

- **屏幕显示**: ≥ **150 DPI**
- **打印输出**: ≥ **300 DPI**（灰度/彩色），≥ **600 DPI**（线图）
- **格式**: 矢量优先（SVG, PDF, EPS），照片用 TIFF/PNG
- **配色**: 避免彩虹/Jet 色图，推荐 viridis 等顺序色板
- **数据-墨水比最大化**: 去掉阴影、3D 效果、粗边框、多余刻度线

### 规则 10: 幻灯片结构流程

1. **标题页** — 标题 + 姓名 + 单位 + 日期 + 会议名
2. **大纲页** — (可选) 帮助听众建立预期
3. **主体 5–8 页** — 每页一个清晰观点
4. **数据可视化页** — 大尺寸、清晰标注的图，一次一页
5. **总结页** — 核心结论；可留在屏幕上供 Q&A
6. **致谢页** — 资助来源、合作者
7. **参考文献页** — 按需

**每类页面的标题应传递信息**，而非仅写 "Results" 或 "Introduction"。

## 三、与当前设计系统的对比

### 当前设计系统概览

| 属性 | 当前值 |
|---|---|
| 主色 | 深蓝 #1a365d |
| 强调色 | 琥珀 #d97706 |
| 正文色 | 石墨 #374151 |
| 背景色 | 钛白 #f7f8fa |
| 标题字体 | Source Han Sans SC (无衬线) |
| 正文字体 | Source Han Serif SC (**衬线**) |
| 标题字号 | 28 pt |
| 正文字号 | 16 pt |
| 布局 | 卡片式网格 + 双栏 |
| 表格样式 | 灰底表头 + 单底线 |
| 图标占位 | 虚线框 + 来源标注 |

---

### 做得好的方面 (3–5 项)

1. **色彩克制且有层次**: 深蓝+琥珀+石墨 三色体系，配合路线专属彩色标签 (D2 蓝/D3 绿/D4 紫/D9 黄)，符合"2–4 色"推荐。琥珀强调色与深蓝对比度高 (约 5.5:1)，视觉有力。

2. **标题-正文比例恰当**: 标题 28 pt : 正文 16 pt ≈ **1.75:1**，落在 1.3:1–1.8:1 的合理范围内。封面标题 36–38 pt 则更好地匹配了"32–40 pt"推荐值。

3. **丰富的内联 SVG 图表**: 路线全景图、温度演化图、N₂ vs Air 对比图等均为矢量格式，数据-墨水比高，无冗余装饰。CSS 变量保证了颜色一致。

4. **卡片式布局系统**: Card-grid 配合 `cols-2/3/4/5` 实现了灵活的内容分块，每块有 `card-header` + `card-body` 清晰层次，白底 + 浅灰边框保持了干净外观。

5. **页脚统一信息**: 每页底部统一放置了数据来源和页码，这在学术汇报中是加分项，体现严谨性。

---

### 可改进的方面 (3–5 项, 供 Task 10 打磨)

1. **正文使用衬线字体 → 改用无衬线**
   - 当前 `--font-body: 'Source Han Serif SC', 'Noto Serif SC', serif`
   - **问题**: 投影环境下衬线小字 (13–16 pt) 锯齿明显，远距离阅读困难
   - **建议**: 正文改用无衬线 (如 Source Han Sans SC 或 Noto Sans SC)，保留衬线仅在标题/引用等少量场景；或正文/标题统一使用同一无衬线家族，用字重 (700 vs 400) 区分层级

2. **标题字号偏小 (28 pt)**
   - **问题**: 学术演示推荐标题 32–40 pt，当前 28 pt 是下限值。14 页以上内容的细长页面中标题视觉权重不够
   - **建议**: 非封面页标题提升至 **32–34 pt**，并相应调整正文至 **18–20 pt**，保持比例同时提升整体可读性

3. **表格样式可向三线表靠拢**
   - 当前 `.data-table` 使用灰底表头 + 单底线，接近三线表但尚有差距
   - **建议修改**:
     - 移除 `background: var(--light-gray)` 表头底色
     - 顶线改为 `border-top: 2px solid var(--deep-blue)` (与底线呼应)
     - 表头下线改为 `border-bottom: 1px solid #9ca3af` (中等粗细)
     - 移除表格内部竖线（当前已无竖线，很好）
     - 表体行交替可用极淡灰 (`#f9fafb`) 辅助浏览，不用全灰底

4. **图占位框的标注信息不标准**
   - 当前 `.placeholder-source` 使用 "来源: PAPER-XXX Fig.X | 位置: PDF 第 N 页" 格式，实用但不符合学术演示的图注规范
   - **建议增加正式图标题仓**: 在占位框下方或内部增加 "Figure X. 自解释描述" 格式的标题行 (14 pt, 比正文小 1 pt)，与学术出版风格一致。当前来源标注可作为补充信息保留。

5. **色彩无障碍性检查未落实**
   - 当前路线颜色 (D1 灰 #6b7280, D2 蓝 #3b82f6, D3 绿 #10b981, D4 紫 #8b5cf6, D9 黄 #f59e0b) 在色盲模拟下可能混淆 (蓝-紫相近，绿-灰在灰度下相近)
   - **建议**: 为每条路线在灰度下的区分度做检查；考虑为路线标签额外添加图标/线型区分，使颜色不是唯一区分维度
   - 具体: 灰度模式下 D2 蓝 (#3b82f6 → 约 30% 灰) 与 D4 紫 (#8b5cf6 → 约 25% 灰) 接近，可微调其一

6. **Section tag 字号过小 (8 pt)**
   - **问题**: 当前左上角色块标签 `font-size: 8pt`，投影后几乎不可读
   - **建议**: 提升至 **10–11 pt**，或使用更短的标签（如用 "0a 文献综述" 替代长标签，或仅在封面/章节页使用）
