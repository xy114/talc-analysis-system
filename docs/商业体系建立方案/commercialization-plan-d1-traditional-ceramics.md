## 1. 价值主张

**一句话定位：**
黑滑石经 800-1000°C 空气煅烧增白→直接作为传统陶瓷/耐火材料/功能性填料的低成本基础原料——是五条路线中技术最成熟、可最快变现、消化矿区最大量原料的"基本盘"路线。

**解决的痛点：**
1. **白滑石资源枯竭**：中国高白度天然白滑石储量有限（辽宁海城/广西龙胜等），价格 500-2000 元/吨且逐年上涨。黑滑石煅烧后白度可达 90-95%——可部分替代白滑石在陶瓷/填料市场的应用
2. **传统煅烧的"路径依赖"**：广丰黑滑石目前 90%+ 的应用就是煅烧增白→卖白粉→中低端陶瓷——但煅烧温度/时间完全凭经验，缺乏科学优化。PAPER-005 Meng 2022 的四阶段热演化数据可系统优化煅烧制度
3. **矿区 10 亿吨储量需要"量"的出口**：D2/D3/D4 三条高值路线加起来最多消化 ~35% 的矿区产出——剩余 65%+ 需要 D1（量大）+ D9（量大）来消化

**对标竞品：**

| 对标 | 白度 | 价格 (元/吨) | 储量 | 弱点 |
|------|:--:|:---------:|------|------|
| 辽宁海城白滑石 | 90-97% | 800-2000 | 濒临枯竭 | 价格看涨/供应不稳 |
| 广西龙胜白滑石 | 88-95% | 500-1500 | 中等 | 品位下降 |
| **D1 煅烧黑滑石** | **88-95%** | **300-800** | **10 亿吨** | 需煅烧能耗/白度略低于天然白滑石 |
| 煅烧高岭土 | 85-95% | 1000-3000 | 丰富 | 成分不同/不能完全替代 |

## 2. 技术路线

### 2.1 核心工艺

```
黑滑石原矿 → 破碎(-200目) → 空气煅烧(800-1000°C, 2-4h)
    ├─ 165-580°C: 有机碳氧化→CO₂↑ (白度提升的关键阶段)
    ├─ 800-900°C: 脱羟基+无定形MgO/SiO₂生成 (活化窗口)
    └─ 900-1000°C: 相转变→顽火辉石+方石英
         ↓
    煅烧白滑石产品:
    ├─ 陶瓷级 (900°C): 白度>90%, 用于陶瓷墙地砖坯体/釉料
    ├─ 填料级 (850°C): 白度>88%, 用于塑料/橡胶/涂料/造纸填料
    ├─ 耐火级 (1000°C): 顽火辉石质, 耐火度>1500°C
    └─ 活化填料级 (800°C): 高比表面积(15 m²/g), 用于复合材料界面增强
```

### 2.2 关键工艺瓶颈

| 瓶颈 | 方案 |
|------|------|
| 煅烧能耗成本（~300-500 元/吨，占总成本 40-50%） | 利用 PAPER-005 热演化数据精确控温→避免过度煅烧；余热回收(烟气→预热进料) |
| 有机碳含量波动→白度批次不一致 | 矿区碳品位分级→高碳矿去 D3，低碳矿来 D1 |
| 石英杂质（5-15%）→限制高端填料应用 | 气流分级去粗石英；或碱洗除硅（成本增加~100 元/吨） |

  ├─ 变现速度: ⭐⭐⭐⭐⭐ (最快——现有需求/现有客户/只需煅烧)

## 4. 竞争分析

### 4.1 直接竞争者

| 竞争者 | 材料 | 年供应量 | 价格 (元/吨) | 优势 | 劣势 |
|--------|------|:------:|:---------:|------|------|
| **辽宁海城白滑石** | 天然白滑石 | ~150 万吨 | 800-2000 | 白度高(>95%)+免煅烧 | 储量枯竭+价格看涨 |
| **广西龙胜白滑石** | 天然白滑石 | ~80 万吨 | 500-1500 | 价格较低+供应稳定 | 白度(88-93%)低于海城 |
| **煅烧高岭土** | 偏高岭土 | ~300 万吨 | 1000-3000 | 白度高+通用性强 | 价格高+成分不同 |
| **重钙/轻钙** | CaCO₃ | ~2000 万吨 | 200-800 | 价格极低+量大 | 耐酸性差+不耐高温 |

### 4.2 D1 的竞争护城河

```
🏰 护城河 1: "10亿吨储量 = 百年供应安全"
白滑石是中国稀缺资源(储量<5000万吨)→客户有供应安全焦虑
→ D1 煅烧黑滑石可以承诺"百年稳定供应"→对大型陶瓷/塑料客户极具吸引力

🏰 护城河 2: "活化态填料"——超越传统滑石的性能
传统白滑石是惰性填料(无化学活性, 比表面积~3-8 m²/g)
D1 的 800°C 活化态产品: 比表面积~15 m²/g + 表面-OH活性位点
→ 可以"功能性填料"定价(1500-3000 vs 普通填料 500-800 元/吨)

🏰 护城河 3: D1 为 D2/D3/D4 提供"客户入口"
向陶瓷厂销售 D1 产品→建立客户关系→再推荐 D2(5G陶瓷粉)/D4(抗菌粉)
→ D1 的客户渠道为高值路线提供"获客成本为零"的交叉销售机会
```

## 6. 放大路线图

### 6.0 TRL 现状：TRL 7-8

```
TRL 7-8 证据:
✓ 广丰黑滑石已有大规模煅烧增白的工业实践(数十万吨级)
✓ 陶瓷厂已大量使用煅烧黑滑石(马可波罗/东鹏等)
✓ 煅烧回转窑技术完全成熟(水泥/石灰/高岭土行业百年经验)

✗ 产品分级体系未建立(所有产品混为一等)
```

### 6.1 第一阶段：TRL 7→8（科学化煅烧制度, 6-12 月）

| 维度 | 内容 |

| **预算** | ~200 万元（工业回转窑中试 + 检测 + 人力） |

**里程碑**：
- [ ] 工业条件下四阶段控温煅烧验证（利用 PAPER-005 的 165°C / 580°C / 800°C / 900°C 关键温度节点优化升温速率和保温段）
- [ ] 10 批次产品白度/比表面积/物相一致性验证
- [ ] 产品分级标准制定（陶瓷/填料/耐火/活化四级的白度/细度/比表面积/吸油值指标界限）

## 10. 与其他路线的协同

| 路线 | 协同维度 | 具体方案 |
|------|---------|---------|
| **D2 (5G陶瓷)** | 原料分流 | D1 用最低品位矿→D2 用高品位矿 |
| | 客户渠道 | D1→陶瓷厂供货→D2→同一客户推荐 5G 陶瓷粉体（交叉销售，获客成本≈0） |
| | 产线复用 | 同一回转窑可切换 D1(空气煅烧)↔D2(预煅烧) |
| **D3 (N₂碳复合)** | 碳态度互补 | D1 煅烧去碳↔D3 保留碳。D1+D3 联合碳排放可趋近"碳中性" |
| **D4 (抗菌)** | 客户渠道 | D1→涂料/塑料厂供货→D4→同一客户推荐抗菌填料 |
| **D9 (酸浸)** | 原料量分流 | D1(65%)+D9(30%) 共同消化矿区主体产量。D9 MgO + D1 耐火级→混配高纯镁质耐火砖 |

> **D1 = 五路线的"底座"**——为 D2/D3/D4 提供客户渠道(交叉销售)+ 规模采购议价力(矿区谈判)+ 现金流(稳定利润)+ 基础设施共享(煅烧窑/公用工程)。利润率最低，但在多路线 Portfolio 中创造最大的**间接受益**。
