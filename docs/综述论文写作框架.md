# 黑滑石高值化综述论文 · 写作框架

> 写给：大一本科生，第一次写论文，不知道论文长什么样、从哪下手
> 目标期刊：《材料导报》或《硅酸盐通报》（中文核心）
> 字数：6000-8000 字 | 参考文献：30-50 篇

---

## 一、一篇论文到底长什么样？

先把一篇论文拆开给你看。**任何学术论文——不管是实验论文还是综述——都由固定的几个「零件」组成。** 就像一辆车一定有轮子、发动机、方向盘一样，论文也一定有下面这些部分，按顺序排：

```
┌────────────────────────────────────────────┐
│  标题 (Title)          ← 一句话说清楚你研究了什么   │
│  作者 + 单位            ← 你和导师的名字            │
├────────────────────────────────────────────┤
│  摘要 (Abstract)        ← 全文的浓缩版，200-300字   │
│  关键词 (Keywords)      ← 3-8个核心术语             │
├────────────────────────────────────────────┤
│  §1 引言 (Introduction) ← 背景：为什么这个题目重要  │
├────────────────────────────────────────────┤
│  §2 正文第1章           │                         │
│  §3 正文第2章           │  ← 论文的主体，           │
│  §4 正文第3章           │    按逻辑分章节展开        │
│  §5 ...                 │                         │
├────────────────────────────────────────────┤
│  §N 结论 (Conclusion)   ← 总结：我们知道了什么       │
├────────────────────────────────────────────┤
│  参考文献 (References)  ← 你引用的所有论文清单       │
└────────────────────────────────────────────┘
```

**每个部分的作用用一句话说清楚：**

| 部分 | 一句话作用 | 读者是谁 |
|------|-----------|---------|
| **标题** | 让读者 3 秒内判断「这篇要不要读」 | 所有人 |
| **摘要** | 让读者 1 分钟内了解全文核心内容 | 赶时间的同行 |
| **引言** | 告诉读者「为什么这个研究值得做」 | 不太了解你方向的研究者 |
| **正文** | 你的论证过程——数据、分析、推理 | 想深入了解的同行 |
| **结论** | 把全文浓缩成几条「带回家的信息」 | 略读了全文的人 |
| **参考文献** | 让读者能追溯你引用的原始出处 | 做相关研究的人 |

**综述论文和实验论文的区别**：

| | 实验论文 | 综述论文 |
|------|---------|---------|
| 数据来源 | 你自己做实验得到的 | 别人已经发表的 |
| 你要做什么 | 报告新发现 | 把已有发现组织成一个有逻辑的故事 |
| 核心能力 | 实验设计 + 数据分析 | 文献梳理 + 逻辑组织 |
| 典型章节 | 实验方法、结果与讨论 | 按主题/路线/年代 分章节 |

你写的是综述——**你的工作不是做实验，而是把别人已经发表的几十篇论文读透，然后用自己的逻辑把它们串成一个故事。**

## 二、按什么顺序写？（这条最重要）

**绝对不要从标题开始写，更不要从引言开始写。**

你从来没写过论文，如果打开一个空白 Word 文档开始写「引言」，你会盯着光标发呆两个小时。因为你根本不知道后面写了什么，你怎么知道前面该引什么？

**正确的写作顺序是从你最确定的、最有把握的部分开始写。** 就好比拼拼图，先把中间图案最清晰的几块拼好，边框最后补。

```
写第1轮：正文主体（你最有把握的章节）
    ↓
写第2轮：讨论/展望（在主体基础上提炼观点）
    ↓
写第3轮：引言（现在你知道你写了什么，才知道要引什么）
    ↓
写第4轮：摘要 + 结论（全文写完后提炼才有意义）
    ↓
写第5轮：打磨标题（最后）
```

**对于你的这篇综述，具体写作顺序是：**

```
第1步：§2 热演化基础      ← 你最熟悉的，PAPER-005 数据很清楚
第2步：§3 去碳路线         ← 数据多，好写
第3步：§4 用碳路线         ← 你的核心创新点
第4步：§5 两条路径对比     ← 需要前两章都写完才能对比
第5步：§6 研究空白与展望   ← 基于前文提炼
第6步：§1 引言             ← 最后写！现在你才知道你要引什么
第7步：§7 结论 + 摘要      ← 全文写完后写
```

**为什么这个顺序能让你不卡壳**：
- 第 1 步的 §2 是纯描述已有研究结果，你不需要创造任何新观点，就是「Meng 2022 做了什么实验，发现了什么」。这是最容易写的，适合热身。
- §3 和 §4 是分路线描述，每一段都是「某论文做了什么，结果是什么，局限是什么」——结构重复，越写越快。
- 写完 §3 和 §4 之后，你对材料的理解已经远超写引言时的你，这时候再写引言，你知道要引什么、要铺垫什么。

## 三、每个部分具体怎么写？

下面以你自己的论文为例，逐节说明**写什么、怎么写、写多少**。

---

### §2 热演化基础（~1200 字）→ 第 1 步写这个

**目的**：给读者建立基础知识——黑滑石加热后会发生什么。

**怎么写**：这一节不要发表你的观点，纯描述已有研究结果。结构：

- **§2.1 四阶段热演化模型**：Meng 2022 把黑滑石从室温加热到 1000°C，用 TG-DSC 和 in-situ XRD 跟踪变化，发现分四个阶段。你把四个阶段说清楚（每个阶段什么温度区间、发生了什么事），然后点出一个关键结论——800-900°C 是「活化窗口」。
- **§2.2 气氛的作用**：吴小文 2017 发现同样的黑滑石，在 N₂ 和空气里烧结果完全不同。用定量数据说明气氛的影响有多大。

**写作难度**：⭐（最低，纯描述已有文献）

**写多少**：3-5 段

---

### §3 「去碳」路线（~2000 字）→ 第 2 步写这个

**目的**：总结目前工业界的主流做法——把碳烧掉。

**怎么写**：逐个路线描述，每条路线都是「原理 → 关键数据 → 产品 → 局限」。

- **§3.1 D1 传统煅烧增白**：空气烧 → 碳变 CO₂ 跑了 → 白度提升 → 产品是低值白色填料（~500-650 元/吨）
- **§3.2 D4 抗菌材料**：先烧掉碳 → 酸活化溶出 Mg²⁺ → 植入 Zn²⁺ → 抗菌功能材料
- **§3.3 D9 酸浸分离**：纯氧预脱碳 → HF+HCl 混合酸浸 → 分离 Mg 和 Si 高纯产品
- **§3.4 小结**：三条路线的共同逻辑——碳 = 需要花钱去除的杂质

**写作难度**：⭐⭐（结构重复，每条路线写法一样）

**写多少**：6-8 段

---

### §4 「用碳」路线（~1500 字）→ 第 3 步写这个

**目的**：介绍你论文的核心创新视角——把碳留下来用。

**怎么写**：这是你论文最重要的章节。重点写 D3 路线（N₂ 气氛下碳石墨烯化）。

- **§4.1 原位石墨烯化的实验证据**：吴小文 2017 做了 N₂ 煅烧黑滑石，TEM 看到了多层石墨烯，性能数据（抗弯 +43%，导电率 ×10）。解释机理——为什么 N₂ 气氛下碳不走氧化而走石墨化。
- **§4.2 与去碳路线的本质区别**：碳从「杂质」变成「功能组元」，这就是为什么 D3 产品价格（5000-20000 元/吨）比 D1（500-650 元/吨）高 10-40 倍。
- **§4.3 其他矿物的交叉验证**：高岭土、蒙脱土也有类似的「矿物自身碳源→石墨烯」的研究，说明这个思路有更广泛的材料科学基础。

**写作难度**：⭐⭐⭐（需要论证你的核心观点，比纯描述难）

**写多少**：5-7 段

---

### §5 两条路径的系统对比（~1000 字）→ 第 4 步写

**目的**：把 §3 和 §4 放在一起做对比，论证「碳态度」决定了路线分叉。

**怎么写**：从技术和经济两个维度做定量对比。建议在这里放一张对比表。

- **§5.1 技术维度**：原料要求、工艺复杂度、设备投入、产品种类
- **§5.2 经济维度**：产品价格区间、市场规模趋势
- **§5.3 推广意义**：不只黑滑石，含碳层状硅酸盐矿物都可以从这个角度重新审视

**写作难度**：⭐⭐（前面两章写完，这章就是归纳对比）

**写多少**：3-4 段

---

### §6 研究空白与未来方向（~800 字）→ 第 5 步写

**目的**：诚实指出目前还不知道什么、接下来该做什么。

**怎么写**：不要说空话（「还需进一步研究」——这种话等于没说）。给具体的、可检验的方向。

- **§6.1 最紧迫的实验数据缺口**：N₂ 下各温度的 Raman 数据、碳含量与石墨烯化程度的关系
- **§6.2 工程放大问题**：从管式炉（克级）到回转窑（百吨级）的挑战
- **§6.3 多路线联用的可能**：去碳和用碳不是互斥的，可能可以结合

**写作难度**：⭐⭐（批判性思维，但不需新材料）

**写多少**：3-4 段

---

### §1 引言（~800 字）→ 第 6 步写

**目的**：让不熟悉黑滑石的读者理解为什么要关注这个题目。

**怎么写**：引言的三段式结构（几乎所有论文的引言都是这个套路）：

1. **第一段**：黑滑石是什么、中国有多少、现在用来干什么（低端填料）→ 核心矛盾：储量巨大 vs 价值极低
2. **第二段**：碳是关键变量——黑滑石「黑」是因为含有机碳，传统做法是把碳烧掉（去碳），但碳能不能留下来用？→ 引出你的核心论点
3. **第三段**：本文怎么组织的——第二节建立热演化基础，第三节讨论去碳路线，第四节讨论用碳路线，第五节对比，第六节展望

**写作难度**：⭐⭐⭐（虽然最后写，但引言决定了审稿人的第一印象）

**写多少**：4-5 段

---

### §7 结论（~300 字）→ 第 7 步写

**目的**：把全文浓缩成几条「读者应该带走的信息」。

**怎么写**：用编号列表，每条一句话。不要引入新信息，不要重复讨论里的细节。结论是全文的「干货提炼」。

**写作难度**：⭐（全文写完后提炼，不难）

**写多少**：4-5 条，每条 1-2 句

---

### 摘要（~200-300 字）→ 最后最后写

**目的**：让读者 1 分钟决定要不要读全文。

**怎么写**：摘要 = 1 句背景 + 1 句问题 + 2-3 句核心内容 + 1 句结论。写完结论后写摘要，因为你已经知道全文的精华是什么了。

**写作难度**：⭐⭐⭐（浓缩最难）

## 四、写作过程中会遇到的几个坑

### 坑 1：打开空白文档不知道写什么

**解决方案**：不要从零写。先把你提纲里「段 1」对应的论文数据整理成笔记（比如把 PAPER-005 的四阶段数据整理成一段话），然后你就是在「翻译 + 组织」已有材料，而不是凭空创造。

### 坑 2：写着写着不知道自己在论证什么

**解决方案**：每写完一个小节，回头读一遍，问自己：「这段话在回答什么问题？」如果答不上来，说明跑偏了。

### 坑 3：引用不知道怎么标注

**解决方案**：初稿阶段不用担心格式。引用某篇论文的数据时，直接在句末用括号写 `[PAPER-005]` 或 `[Meng 2022]`。等全文写完后，统一把所有引用替换为 `[1]`、`[2]` 序号格式，然后在文末按序号列出完整的参考文献信息。**先写内容，最后统一调格式。** 千万不要一边写一边调格式，那会让你一个字都写不出来。

### 坑 4：觉得自己的话没有论文原文写得好，不敢改写

**解决方案**：**绝对不能直接抄原文。** 综述的核心工作就是把别人的结果用自己的话重新说一遍。技巧：读原文 → 把论文合上 → 用自己的话写出来 → 再打开论文对一下数字有没有写错。如果你觉得原文某句话特别好必须保留，用引号并注明出处。

## 六、和导师第一次讨论你该带什么

1. **本文档的「二、按什么顺序写」和「三、每个部分怎么写」** —— 让导师知道你对论文结构有概念
2. **一句话**：「碳不是杂质，是碳资源——对碳的态度决定了黑滑石高值化的全部路径」
3. **一个问题**：「您觉得这个论点立得住吗？有没有更合适的框架？」

**不要让导师替你想提纲——带着自己的方案去讨论。** 导师看到一个大一学生能拿出完整的写作框架和核心论点，会认真对待你的讨论请求。

> 以下是各章节的段落级提纲。建议**先读完上面五节，理解了论文结构和写作顺序之后，再看这个提纲**。写作时把这个提纲放在旁边对照着写。

### 论文标题（建议）

### §1 引言（~800 字）→ 最后写

**§1.1 黑滑石：被低估的巨量资源（2 段）**

- 段 1：黑滑石的基本定义（2:1 T-O-T 层状硅酸盐，含 1-2% 有机碳）+ 中国资源禀赋（广丰 >10 亿吨远景资源量，探明 ~1.8 亿吨）→ 引用 [1] + [2]
- 段 2：当前利用现状——大部分用于低端填料（原矿 ~200 元/吨），高值化率极低 → 核心矛盾：储量巨大 vs 价值极低

**§1.2 碳：被忽视的变量（2 段）**

- 段 1：黑滑石「黑」的来源——层间类石墨烯碳层（[1]）→ 传统认知视碳为杂质
- 段 2：**本文核心论点**——碳不是杂质，是碳资源。对碳的态度（去除/保留）导致了两条根本不同的技术路径。已有综述未从这个角度组织

**§1.3 本文结构（1 段）**

- 段 1：简述全文逻辑

---

### §2 黑滑石热演化基础（~1200 字）→ 第 1 步写

**§2.1 四阶段热演化模型（3 段）**

- 段 1：Meng 2022 [3] 的四阶段模型——30-165°C 脱水 / 165-580°C 碳氧化 / 800-900°C 脱羟基 / 900-1000°C 相转变。TG-DSC + in-situ XRD + 比表面积数据，15 个温度点
- 段 2：关键发现——800-900°C 是「活化窗口」，无定形 SiO₂/MgO 含量和比表面积达峰值
- 段 3：Huang 2025 [4] 的反向验证——1100°C 纯氧脱碳 99.93%，证明碳的热化学行为高度可控

**§2.2 气氛变量的决定性作用（2 段）**

- 段 1：吴小文 2017 [5] 首次在 N₂ 和 Air 下对比煅烧，抗弯 +43%，导电率 ×10
- 段 2：构建「温度-气氛-产物」分析框架——温度决定反应阶段，气氛决定碳去向

---

### §3 「去碳」路线（~2000 字）→ 第 2 步写

**§3.1 D1 传统煅烧增白（2 段）**

- 段 1：原理——空气煅烧 → 碳氧化为 CO₂ → 白度 >90%（引用 [3] 碳氧化窗口 165-580°C 数据）
- 段 2：局限——产品为低值白色填料（~500-650 元/吨），传统应用市场萎缩（[6]）

**§3.2 D4 抗菌材料（2 段）**

- 段 1：裘林莉 2017 [7]——空气煅烧去碳 → 酸活化溶出 Mg²⁺ (352 mmol/L) → Zn²⁺ 植入晶格重建 → 抗菌功能
- 段 2：评价——本质仍是「先去碳再功能化」，碳是必须清除的障碍

**§3.3 D9 酸浸分离（2 段）**

- 段 1：Huang 2025 [4]——1100°C 纯氧脱碳 99.93% → HF+HCl 混合酸浸 → Mg 浸出率 97.98%，SiO₂ 纯度 99.92%
- 段 2：局限——HF 环保壁垒 + 高 CAPEX，碳被完全氧化浪费

**§3.4 去碳路线的共同特征（1 段）**

- 段 1：三条路线的共同逻辑——碳 = 需要花钱去除的杂质。这是工业界主流思维

---

### §4 「用碳」路线（~1500 字）→ 第 3 步写

**§4.1 D3 N₂ 气氛原位石墨烯化（3 段）**

- 段 1：吴小文 2017 [5] 核心发现——N₂ 气氛下层间碳部分石墨化，TEM 确证多层石墨烯。抗弯 +43%，导电率 ×10，导热 +0.62 W/(m·K)
- 段 2：机理——结合 [3]，N₂ 抑制碳氧化窗口（165-580°C），碳在活化窗口（800-900°C）发生 sp² 有序化重排
- 段 3：碳源特殊性——原位 vs 外加，单步法成本大幅降低

**§4.2 D3 与去碳路线的本质差异（2 段）**

- 段 1：碳的角色翻转——从「杂质」到「功能组元」。产品价格差 10-40 倍
- 段 2：当前局限——仅 [5] 一篇实验验证，缺乏 Raman ID/IG、温度-石墨烯化曲线、percolation 阈值

**§4.3 其他矿物的交叉启示（1 段）**

- 段 1：高岭土/蒙脱土等也有「矿物自身碳源→石墨烯」研究，黑滑石是未被充分探索的原料

### §5 「碳态度」决定论（~1000 字）→ 第 4 步写

**§5.1 技术维度对比（2 段）**

- 段 1：五维度框架——原料要求、工艺复杂度、设备投入、产品种类、产品价值
- 段 2：建议制作「去碳 vs 用碳」对比表（核心插图）

**§5.2 经济维度对比（2 段）**

- 段 1：价格阶梯——引用 [6] 全球滑石市场数据
- 段 2：市场规模趋势——传统应用萎缩，功能填料增长（塑料占美国滑石消费 32%）

**§5.3 推广意义（1 段）**

- 段 1：不只黑滑石——含碳层状硅酸盐矿物可能都可以从「去碳」转向「用碳」

### §6 研究空白与未来方向（~800 字）→ 第 5 步写

**§6.1 最紧迫的科学问题（2 段）**

- 段 1：按优先级排列的实验数据缺口——
    1. N₂ 下各温度的 Raman ID/IG 定量数据
    2. N₂ vs Air 同温度对照实验（4 温度 × 2 气氛 × 3 重复 = 25 样品）
    3. 碳含量与石墨烯化程度的定量关联
    4. 导电 percolation 阈值
- 段 2：这些实验将检验核心假设——碳含量越高的黑滑石，N₂ 下石墨烯化程度是否越高？

**§6.2 工程放大挑战（1 段）**

- 段 1：管式炉（克级）→ 回转窑（百吨级）的 N₂ 气氛放大——动密封、N₂ 成本、安全性

**§6.3 多路线协同（1 段）**

- 段 1：去碳与用碳不互斥——D3 + D9 联用可能实现更高全资源利用率

---

### §7 结论（~300 字）→ 第 7 步写

1. 黑滑石中有机碳的热化学行为是决定高值化路径的核心变量
2. 「去碳」路线（D1/D4/D9）技术成熟但产品同质化，价值天花板低
3. 「用碳」路线（D3）利用自身碳原位石墨烯化，产品价值提升 10-40 倍，但数据极度稀缺
4. 从「去碳」到「用碳」的认知翻转，对更广泛的含碳层状硅酸盐矿物具有启示意义
