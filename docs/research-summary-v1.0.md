- **目标原料**：黑滑石（低品位滑石，含有机碳和石英杂质，中国探明储量近 10 亿吨）

## 二、已入库论文详解

### 2.1 PAPER-002: 杨华明综述 (2023)

**核心价值**：全技术路线图的战略框架，约90篇参考文献。

**关键数据**：
- 黑滑石全球储量：中国探明近10亿吨（广丰为核心）
- 6 矿区化学成分表（广丰：SiO2 55.84%, MgO 27.86%, LOI 11.14%）
- **煅烧四阶段反应方程**（750-1150°C）——所有热化学路线的理论基础
- N₂气氛烧结增强效应：导电率提升10倍、抗弯强度 51.8 MPa
- 技术方向全覆盖：陶瓷/填料/锂电碳材料/光催化/电磁波吸收/抗菌/化妆品

**评分**：技术7 / 经济4 / 市场8 / 放大5 → 综合 65.0

### 2.2 PAPER-004: 刘子峥 5G微波介质陶瓷 (2024)

**核心创新**：首次以黑滑石矿物为单一原料制备5G微波介质陶瓷。

**关键数据**：
- 原料：广丰黑滑石，XRF: SiO2 65.64%, MgO 23.39%, LOI 5.79%
- 工艺：800°C预煅烧2h → 湿法球磨8h → 1325°C烧结4h（固相烧结法）
- **最佳性能 @1325°C**：εr=5.11, Q×f=56782 GHz, τf=-17.85×10⁻⁶/°C
- 体积密度最大 2.97 g/cm³
- 物相：MgSiO3顽火辉石 + SiO2方石英两相共存

**对标分析**：εr 介于纯 MgSiO3(6.71) 和纯 SiO2(3.52) 之间——黑滑石天然就是两者的复合体
**短板**：Q×f 偏低（纯 MgSiO3 ~120000 GHz）——主因微量杂质恶化介电损耗
**改善方向**：原料提纯降Fe → 提升Q×f；添加TiO2/CaTiO3 → 调τf近零；N₂气氛烧结 → 提升性能

**评分**：技术8 / 经济6 / 市场9 / 放大7 → 综合 **75.0** 🥇

### 2.3 PAPER-003: 裘林莉 酸活化晶格重建 (2017)

**核心创新**：首次在黑滑石体系验证"酸活化溶出Mg²⁺→Zn²⁺植入重建晶格"的技术路径。

**关键数据**：
- 原料：广丰黑滑石煅烧产物（白色粉末，顽火辉石+硅灰石+石英）
- 工艺：(1) 2M HCl, 80°C, 4-32h → (2) 2M ZnSO4, 80°C, 4-16h
- Mg²⁺溶出量 16h 达 352 mmol/L
- 晶胞参数 c 从 0.51052→0.52587 nm（32h酸活化，八面体膨胀）
- Zn²⁺重建后 FWHM 从 0.280 降至 0.199（恢复优于原矿0.214）
- **关键发现**：石英耐酸性——酸活化选择性溶蚀顽火辉石而保留石英骨架
- 抗菌：载Zn产物对大肠杆菌和金黄色葡萄球菌生长有明显抑制

**平台潜力**：Zn²⁺可替换为 Cu²⁺（抗菌）、Ag⁺（高效杀菌）、稀土离子（发光/催化）
**定位**：高利润功能材料，非大宗消纳路线

**评分**：技术7 / 经济5 / 市场6 / 放大5 → 综合 57.5

### 3.1 论文概要

| 字段 | 值 |

| 作者 | Meng Yuhang, Xie Weimin, Wu Haiyan, Tariq S.M., Yang Huaming |

| 研究类型 | 基础实验研究 |

### 3.2 核心发现

**黑滑石热演化四阶段（30-1000°C）**：

| 阶段 | 温度区间 | 过程 | 质量损失 | 关键证据 |
|------|----------|------|----------|----------|
| I | 30-165°C | 脱水（表面吸附水） | 0.32% | MS m/z=18 在152°C→0 |
| II | 165-580°C | 有机碳氧化分解 | 0.65% | MS m/z=44 在600°C峰值；d(006) 从0.313→0.315nm（CO2逸出撑开层间距） |
| III | 800-900°C | 脱羟基 | (含于总失重) | 相对结晶度 CI 从75.8%→51.0%；FTIR O-H 3676→3666 cm⁻¹ |
| IV | 900-1000°C | 相转变（滑石→顽火辉石） | (含于总失重) | T950 CI=27.3%，T1000 滑石反射完全消失 |

**580-1000°C 总失重 7.02%**（含白云石分解+滑石脱羟基+相转变）

### 3.3 区别于普通滑石的关键差异

| 特性 | 普通滑石 | 黑滑石（本论文发现） |
|------|----------|----------------------|
| 脱羟基起始温度 | 850-900°C | **800°C**（更低！） |
| 热稳定性 | 较高 | **较低**——有机碳和杂质降低了热稳定性 |
| 950°C 结晶度 | 较高 | CI 仅 27.3%，大量无定形 SiO2/MgO |

### 3.4 对研究体系的增强价值

> ⭐ **这是论文 #7 综述中提到的「煅烧四阶段反应」的原始基础研究论文**。两者来自同一课题组（杨华明），构成"综述框架↔实验基础"的完整证据链。

对每条商业路线的直接影响：

| 路线 | 论文 #3 提供的指导 |
|------|-------------------|
| **D1 传统陶瓷** | 增白的最佳煅烧温度应 >950°C（有机碳完全氧化 + 相转变完成）；但超过 1000°C 时比表面积从 18.8→5.3 m²/g 急剧下降——烧结活性降低 |
| **D2 5G微波介质陶瓷** | 预煅烧 800°C 是合理选择——此时脱羟基刚开始，层状结构部分保留，有机碳已去除。若预煅烧 >900°C 则完全转为顽火辉石+无定形，可能改变后续烧结行为 |
| **D4 抗菌功能材料** | **800-900°C 是"活化窗口"**：无定形 SiO2/MgO 含量增加 → 表面活性提高 → 有利于酸活化。论文 #11 用的煅烧产物正是这个窗口的产品 |
| **D3 N₂气氛材料** | 有机碳在 165-580°C 氧化——N₂气氛下可抑制氧化，使碳保留并石墨化。预煅烧温度应 <580°C |
| **酸浸分离路线** | 无定形 MgO 含量随温度升高而增加（图 6b）→ 酸浸提取 Mg 应选择 ≥900°C 的煅烧产物 |

**两个关键工程参数**：
1. **Zeta 电位**：T800 表面电位仅为原矿的一半，且几乎不随 pH 变化——这对湿法分散/浆料配方至关重要
2. **颗粒粒径**：800°C 后从 21.94→28.17→32.78 μm（T1000），粒度增大意味着需要球磨控制粒径
