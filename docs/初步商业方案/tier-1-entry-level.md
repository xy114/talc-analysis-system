## 1. 企业画像：什么样的企业是初阶合作对象

| 维度 | 画像 |
|------|------|
| **企业规模** | 年营收 < 5,000 万，员工 < 100 人 |
| **企业类型** | 广丰本地煅烧厂 / 小型陶瓷原料加工厂 / 个体矿业加工户 |

**分析框架——"温度-气氛-产物"三位一体相图**：

```
                   空气气氛                          N₂气氛
                   ────────                        ──────
 30-165°C   脱水（表面水逸出）                脱水（同空气）
165-580°C   有机碳氧化→CO₂↑（碳损失）          有机碳热解→部分石墨化（碳保留）⭐
580-800°C   CaCO₃→CaO+CO₂↑（白云石分解）      同空气 + 碳进一步石墨化
800-900°C   脱羟基→无定形MgO/SiO₂生成           脱羟基 + 碳石墨烯化加速 ⭐ 活化窗口
900-1000°C  相转变→顽火辉石+方石英              顽火辉石 + 原位多层石墨烯 ⭐
           产物: 白色惰性陶瓷填料                产物: 灰色导电/导热功能填料
```

> 这个相图本身就是一篇可发表的理论论文——属于"数据二次分析"（data re-analysis）类型，不需要新实验。

**产出**：温度-气氛-产物相图 + 1篇英文或中文研究论文（目标期刊：*Minerals* 或 *Applied Clay Science*）

### 3.3 Phase 0c：管式炉验证实验（预算 ~5,000 元，2-3 月）

**目标**：用管式炉+N₂瓶+广丰黑滑石原矿→验证 PAPER-006 的关键发现（碳石墨化行为）。

**实验设计（直接复用已有论文的参数）**：

| 参数 | 取值 | 来源论文 |
|------|------|---------|
| 原料 | 广丰黑滑石原矿，200目过筛 | PAPER-004 刘子峥(200目) / PAPER-005 Meng(原矿直接使用) |
| 原料成分参考 | SiO₂ 65%, MgO 27-31%, CaO 0.2-4.2%, 碳 1-2% | PAPER-005 Meng(高MgO) / PAPER-007 Huang(高CaO) / PAPER-004 刘子峥(中等品位) |
| 温度点 | 580°C / 800°C / 900°C / 1000°C（4个关键节点） | PAPER-005 Meng四阶段节点 + PAPER-006 N₂煅烧温度 |
| 气氛 | N₂(99.9%+), 流量50-200 mL/min | PAPER-006 吴小文 |
| 保温时间 | 2h（参考 PAPER-005 Meng保温120min + PAPER-004 预煅烧2h） | PAPER-005/PAPER-004 |
| 升温速率 | 10°C/min | PAPER-005 Meng |
| 样品量 | 5-10g/批次 | 管式炉常规量 |
| 表征手段 | XRD（物相）+ Raman（碳石墨化程度 ID/IG）+ 四探针（电阻率） | PAPER-006未做Raman——这是你的增量贡献 |

**增量创新点**（PAPER-006 论文未做但你可以做的）：
- ✨ **Raman光谱 ID/IG比** —— PAPER-006 只做了XRD+TEM，没有Raman定量石墨化程度。这是你论文的增量
- ✨ **N₂流量梯度实验** —— 不同N₂流量(50/100/200 mL/min)对石墨化程度的影响
- ✨ **碳含量-石墨化程度关联** —— 选取不同碳品位(高碳~2% vs 低碳~0.5%)的黑滑石做对比

**预算拆解**：

| 项目 | 预算 | 说明 |
|------|:---:|------|
| 广丰黑滑石样品 | ~500元 | 网购/联系矿区寄送(5kg足够) |
| N₂气瓶(40L) × 2 | ~800元 | 高纯N₂ 99.999% |
| 管式炉使用费 | ~2,000元 | 若学校有则仅电费；若需外协则按机时计 |
| Raman测试 × 8样 | ~1,200元 | 校外测试 ~150元/样 |
| XRD测试 | ~500元 | 若学校有则免费 |
| 合计 | **~5,000元** | |

**产出**：N₂气氛碳石墨化的Raman定量数据（ID/IG比 vs 温度 vs N₂流量）+ 1篇实验论文（目标期刊：*Ceramics International* 或 *Applied Clay Science*）

## 5. 与导师讨论的关键问题

```
1. 学校是否有马弗炉/管式炉可以用于N₂气氛实验？
   → 如果有 → Phase 0c 可以立即启动
   → 如果没有 → 先做 Phase 0a + 0b（纯文献分析，也能发论文）
   → 参考：PAPER-005 Meng用的设备是NETZSCH STA 449F3(TG/DSC)+PANalytical Empyrean(in-situ XRD)
            如果学校有这些设备，实验质量可以直接对标SCI论文

2. 导师是否有广丰本地煅烧企业的联系人？
   → 如果有 → 可以先做一次企业拜访，了解对方真实需求
   → 如果没有 → 通过论文发表+专利申请建立"专家身份"再接触
   → 参考：PAPER-010 Fang(杭电)已有产业合作方(浙江瓦赞新材料)——说明"D2路线企业合作"已有先例

3. 课题组是否可以安排1-2名研究生参与？
   → 如果有 → 分工: 研究生做实验，本科生做文献+数据分析
   → 如果没有 → 本科毕业设计即可覆盖 Phase 0a-0c
   → 参考：Phase 0c的实验量≈一个本科毕业设计(8-12周实验+4周写作)

4. 导师对哪个路线最感兴趣？
   → D3（石墨烯填料）: 最容易出论文+专利，学术价值高，已有PAPER-006原始验证
   → D2（5G陶瓷）: 论文档次更高（SCI Q1），5篇论文支撑(PAPER-004/10/13/14/15)
   → D1（传统煅烧）: 论文创新性一般，但产业转化最快，已有PAPER-005完整热演化数据

5. 【新增】是否需要先行补充关键缺失论文？
   → 最紧迫缺失：Mao/Song et al. 2025（Ge⁴⁺掺杂MgSiO₃, Q×f=216,880 GHz）
     — 这是目前D2路线中Q×f最高的掺杂策略，对综合分析至关重要
   → 建议：安排1名研究生或本科生通过文献传递/SCI-HUB获取全文
```

## 7. 企业实力评估速查表

### 7.1 企业硬实力评分

| 维度 | 1分 (弱) | 2分 (中) | 3分 (强) |
|------|---------|---------|---------|
| **财务** | 年营收 < 5000万 | 5000万-5亿 | > 5亿 |
| | 无外部融资能力 | 银行信贷可用 | 资本市场/政府背书 |
| **资产** | 租赁厂房/设备 | 自有中小产线 | 自有矿区+大规模产线 |
| **技术** | 无研发人员 | 1-3名技术员 | 独立研发中心 |
| **市场** | 本地客户/代工 | 省级经销商网络 | 全国/出口渠道 |
| **管理** | 家族式/个体户 | 有管理层但非专业化 | 职业经理人+现代企业制度 |

- 5-7 分 → **初阶**合作对象（当前文件）

- 12-15 分 → **高阶**合作对象
