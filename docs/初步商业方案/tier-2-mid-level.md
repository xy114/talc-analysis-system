## 1. 企业画像：什么样的企业是中阶合作对象

| 维度 | 画像 |
|------|------|
| **企业规模** | 年营收 5,000 万 - 5 亿，员工 100-1,000 人 |
| **企业类型** | 省级陶瓷集团 / 中型耐火材料企业 / 化工原料加工企业 / 有技术升级需求的传统矿业公司 |
| **现有资产** | 自有矿山或长期供应合同，多条产线，有质检实验室 |
| **研发能力** | 有 1-3 名技术员，可能与高校有合作经验 |
| **投资能力** | 500-5,000 万元，回收期容忍 2-4 年 |
| **风险承受** | 中等——可承受单条产线失败，但不能影响主营业务 |
| **技术消化** | 可参与联合研发，但核心 IP 需课题组提供 |
| **课题组角色** | 🤝 **联合研发方**：技术方案 + 中试指导 + 人才培养(研究生驻厂) |

### 3.4 D2 中阶详细路径（高壁垒长期回报）——论文数据支撑版

```
投入: ~2,300万 | 回收: ~11月 | 毛利率: ~90%
论文资产: 5篇SCI论文覆盖从基体到掺杂的完整技术拼图

Phase 2a (月0-12): 实验室配方开发+验证
  ├─ 基体配方 — 2条并行路线
  │   ├─ 路线A (MgSiO₃体系): PAPER-004 刘子峥 2024
  │   │   └─ εr=5.11, Q×f=56,782 GHz, τf=−17.85, 1325°C/4h, 800°C预煅烧2h
  │   │   └─ 原料: 广丰黑滑石 SiO₂ 65.64%, MgO 23.39%, Fe₂O₃ 0.05%(低铁优势!)
  │   └─ 路线B (CaMgSiO₄体系): PAPER-008 Xiao 2025
  │       └─ εr=8.14, Q×f=59,212 GHz, τf=−69.84, 1370°C/4h
  │       └─ 原料成本降80%(¥10/kg滑石 vs ¥330/kg高纯MgO+SiO₂)
  │
  ├─ 掺杂策略 — 3条并行路线(按优先级排序)
  │   ├─ Cu²⁺掺杂(优先推荐): PAPER-010 Fang 2024
  │   │   └─ Mg₁₋ₓCuₓSiO₃ x=0.05: Q×f=93,600 GHz(+65% vs PAPER-004), εr=6.2, τf=−33.7
  │   │   └─ 机制: 抑制PEN→CEN相变(解决PAPER-004的核心瓶颈!) / 1年稳定性验证通过
  │   │   └─ 成本优势: CuO ~¥50/kg(极低) + 有产业合作方(浙江瓦赞新材料)
  │   ├─ Co²⁺掺杂(高Q×f备选): PAPER-011 Ullah 2019
  │   │   └─ Mg₁₋ₓCoₓSiO₃ x=0.15: Q×f=145,846 GHz(目前第二高!), εr=8.11, τf=−12(近零!)
  │   │   └─ 劣势: Co₃O₄ ~¥200-300/kg(比CuO贵4-6倍)
  │   └─ Ge⁴⁺掺杂(天花板方案—需补充论文): Mao/Song et al. 2025
  │       └─ Q×f=216,880 GHz(目前全球最高!) — 论文待获取
  │
  ├─ 低温烧结策略: PAPER-009 Bu 2025
  │   └─ Li₂₋ₓMgSiO₄ 1075°C/4h: εr=6.08, Q×f=55,723 GHz
  │   └─ 比PAPER-004的1325°C低250°C! 节能~20%
  │
  ├─ **推荐最优策略: Cu²⁺+Co²⁺共掺杂**
  │   └─ 低Cu(1-3% 相稳定/低成本) + 低Co(5-10% 高Q×f/近零τf)
  │   └─ 目标: Q×f>120,000 + τf<±15 + 烧结温度<1300°C
  │   └─ 参考: PAPER-010 Cu单独Q×f=93,600 + PAPER-011 Co单独Q×f=145,846 → 共掺杂有望>120,000
  │
  └─ 预算: ~100万(试剂+检测+人工, 12个月)

Phase 2b (月12-24): 中试线建设+产品定型
  ├─ 设备: 球磨+喷雾造粒+干压成型+高温烧结炉(1400°C, 气氛可控)
  ├─ 产能: 100吨/年陶瓷粉体 | 预算: ~1,200万
  ├─ 产品系列(基于论文数据):
  │   ├─ MS-L (低介系列, εr=5-6): PAPER-004基体 + PAPER-010 Cu掺杂 → 毫米波天线基板
  │   ├─ MS-M (中介系列, εr=6-8): PAPER-009 Li₂MgSiO₄低温体系 → 滤波器/谐振器
  │   └─ MS-H (高Q系列, Q×f>120,000): PAPER-011 Co掺杂 → 高端谐振器/基站滤波器
  └─ 原料品控: 基于PAPER-005 Meng + PAPER-004 刘子峥 XRF数据→建立原料MgO/Fe₂O₃品位标准

Phase 2c (月24-36): 客户验证+首批订单
  ├─ 送样目标: 华为/中兴/信维通信/灿勤科技/大富科技等5G滤波器/天线厂商
  ├─ 验证周期: 6-12月(客户需做滤波器仿真设计+小批量试产+可靠性测试)
  ├─ 首批订单目标: 10-30吨/年 → 营收500-1,500万(均价~50万/吨)
  └─ 预算: ~500万(产线调试+客户支持+认证费用)

Phase 2d (月36-48): 商业化量产+客户扩充
  ├─ 扩产至500吨/年 → 年营收~2.5亿 | 年毛利~2.25亿
  ├─ 追加投资: ~500万
  └─ 性能天花板: 按Mao/Song 2025的Ge掺杂(Q×f=216,880)→储备下一代产品
```

```
我们出:                            企业出:

  │   └─ 证据: 10篇论文的完整数据      ├─ 现有产线/场地
  ├─ 专利/IP许可                      ├─ 市场渠道/客户关系
  │   └─ 证据: 初阶积累的发明专利      ├─ 供应链管理
  ├─ 技术指导(研究生驻厂)              └─ 政府关系/环评资质
  ├─ 持续研发(下一代产品)
  │   └─ 证据: 掺杂策略迭代路线图
  └─ 检测分析能力

收益分配建议:
  ├─ 初阶试点: 技术服务费(一次性或按阶段) + 论文发表权
  ├─ 中阶合作: 技术许可费(按销售额3-5%) + 联合申请政府项目
  └─ 高阶合作: 技术入股(10-20%) + 联合实验室 + 联合培养
```

## 5. 中阶风险矩阵

| 编号 | 风险 | 概率 | 影响 | 等级 | 论文数据依据与缓解 |
|:---:|------|:---:|:---:|:---:|------|
| **R1** | **企业中途撤资**——企业经营变化导致研发项目被砍 | 中 | 严重 | 🟠 | 缓解: 多企业并行接触；阶段性成果交付；合同约定退出条款。参考: PAPER-010 Fang已有产业方参与→证明D2方向有企业信心 |
| **R2** | **技术放大失败**——实验室→中试线不work | 中 | 严重 | 🟠 | 初步工业验证数据: PAPER-004的固相烧结法本身就是工业成熟工艺→放大风险较低；PAPER-005的800°C活化窗口在PAPER-004的预煅烧中得到验证→实验室与工业的条件连续性较好。D2关键风险在掺杂均匀性(PAPER-010 Cu掺杂在公斤级是否仍均匀？) |
| **R3** | **客户不接受产品**——D2送样被5G厂商拒绝 | 中高 | 严重 | 🔴 | 先用D3验证市场(D3填料客户验证周期1-2月，vs D2陶瓷客户验证6-12月)。D2性能对标: PAPER-004的Q×f=56,782低于纯MgSiO₃(120,000)→需通过掺杂提升至>80,000才能到达客户门槛 |
| **R4** | **D2的τf不达标**——τf= −17.85(PAPER-004)到−189(PAPER-009)，全部为负且偏离大 | 高 | 中等 | 🟡 | PAPER-011的Co掺杂将τf改善至−12(接近±10商业门槛)。策略: 添加TiO₂(τf~+450)或CaTiO₃(τf~+800)正温度系数补偿剂→可将τf调至近零 |
| **R5** | **IP被企业拿走**——企业学会技术后甩开课题组 | 中 | 严重 | 🟠 | 核心参数不一次性交付(分阶段)；专利申请在前合作在后；合同约定技术回授条款。**关键保护**: 掺杂配方(Cu/Co比例)是核心Know-how——只交付"做什么"不交付"为什么" |
| **R6** | **Q×f批次波动**——黑滑石原料微量元素(Fe³⁺等)恶化介电损耗 | 中高 | 中等 | 🟡 | PAPER-004 Fe₂O₃=0.05%(极低！这是广丰矿的优势)；PAPER-005 Fe₂O₃=0.15%(仍低)；PAPER-007 Fe₂O₃未报道(含CaO 4.23%高→可能导致第二相恶化Q×f)。**关键**: 建立原料品位分级制度(MgO>28%+Fe₂O₃<0.1%定为D2级) |

## 6. 企业实力评估速查

| 维度 | 1分 (弱) | 2分 (中) | 3分 (强) |
|------|---------|---------|---------|
| **财务** | 年营收 < 5000万 | 5000万-5亿 | > 5亿 |
| | 无外部融资能力 | 银行信贷可用 | 资本市场/政府背书 |
| **资产** | 租赁厂房/设备 | 自有中小产线 | 自有矿区+大规模产线 |
| **技术** | 无研发人员 | 1-3名技术员 | 独立研发中心 |
| **市场** | 本地客户/代工 | 省级经销商网络 | 全国/出口渠道 |
| **管理** | 家族式/个体户 | 有管理层但非专业化 | 职业经理人+现代企业制度 |

- 5-7 分 → 初阶合作对象 | **8-11 分 → 中阶合作对象** | 12-15 分 → 高阶合作对象

### 企业意愿评估

| 信号 | 解读 |
|------|------|
| ✅ 主动联系课题组/学校 | 有技术升级的内在驱动力 |
| ✅ 愿意派人参加技术讨论 | 不是"甩手掌柜" |
| ✅ 能说清自己的痛点和需求 | 对市场有认知 |
| ✅ 有高校合作经验 | 降低沟通成本和期望管理难度(**参考: PAPER-010 Fang 杭电+浙江瓦赞模式**) |
| ⚠️ 只说"你们做出来我就买" | 可能是套技术方案——要求其承担部分研发成本来验证诚意 |
| ❌ 不愿意签保密协议 | 对方可能不尊重IP——一票否决 |
| ❌ 要求"技术入股但不承担研发费用" | 风险不对等——要求至少承担中试费用的50%+ |

## 7. 中阶进入高阶的转化条件

```
中阶成果检验:
  ├─ 是否有稳定量产的中试线（1年以上连续运行）？
  │   └─ D3: 200吨/年稳定12月+ → 满足 | D2: 100吨/年稳定12月+ → 满足
  ├─ 是否有 ≥ 3 家稳定付费客户？
  │   └─ D3: 填料客户(塑料/涂料/橡胶) → 验证周期短 | D2: 5G器件厂商 → 验证周期长
  ├─ 年营收是否达到 500 万+ 且毛利率 > 50%？
  │   └─ D3: 100吨×5万/吨=500万, 毛利率93% → 满足 | D2: 10吨×50万/吨=500万, 毛利率90% → 满足
  ├─ 团队是否培养了 ≥ 2 名能独立操作的技术骨干？
  │   └─ 研究生驻厂1-2年 + 企业技术员培训
  └─ 是否有政府/产业资本的关注或接触？
      └─ 通过学术会议+专利推介+政府对接→建立初步联系
                                        ↓
                              高阶合作谈判筹码
                                        ↓
                  → 详见 [高阶方案](tier-3-advanced.md)
```
