# Task 9 Report: Academic PPT Design Reference

## Deliverables
- **presentation/DESIGN-NOTES.md** — comprehensive design reference

## Research Summary

Searched 3 web queries covering MRS template design, materials science presentation design, and scientific typography/color best practices. Aggregated guidelines from University of Illinois PHYS 595, University of Paderborn, The Scientist Magazine, IEEE, Purdue, Durham Physics, and other authoritative sources.

## Key Outputs

### 10 Actionable Design Rules
1. Font hierarchy: title 32-40pt, body 24-28pt, ratio 1.3:1 to 1.5:1
2. Slide density: 6x6 rule, one core point per slide, 7-second test
3. Color restraint: 2-4 colors, contrast ≥4.5:1, no color-only distinctions
4. Data figure annotation: axes with units, panel labels (A,B,C), error bars, direct labeling
5. Three-line table standard: 1.5pt top/bottom, 0.75pt header rule, no verticals, no shading
6. Figure captions below (self-contained), table captions above
7. Sans-serif for body (Calibri/Arial), max 2 font families
8. Light background vs dark background room-size decision
9. Resolution: ≥150 DPI screen, ≥300 DPI print, vector preferred
10. Slide structure: message-driven headings, one idea per slide

### Design System Comparison

**Strengths (5 items):**
- Color restraint with deep blue + amber + graphite
- Good title-to-body ratio (~1.75:1)
- SVG inline diagrams with high data-ink ratio
- Card-based layout system with clear hierarchy
- Uniform footer with data sources and page numbers

**Improvements (6 items, prioritized for Task 10):**
- P0: Serif body font → switch to sans-serif for projection readability
- P1: Title 28pt → 32pt minimum
- P2: Table style → three-line standard
- P3: Figure placeholders → add formal captions
- P4: Accessibility check for color blindness
- P5: Section tag 8pt → 10-11pt
